## 🎯 Introduction

This repository is designed as a comprehensive guide for anyone looking to learn shell scripting from the ground up. Whether you're a system administrator, DevOps engineer, or just someone interested in automating tasks on Linux/Unix systems, this collection of tutorials and examples will help you master the fundamentals and advance to real-world scripting scenarios.

---

## ✅ Prerequisites

- Basic understanding of Linux/Unix command line
- A Linux/Unix environment (Ubuntu, CentOS, macOS, or WSL on Windows)
- Text editor (vim, nano, VS Code, etc.)
- Bash shell (version 4.0 or higher recommended)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/bhuvan-raj/ShellScripting-from-Scratch.git
cd ShellScripting-from-Scratch
```

### Make Scripts Executable

```bash
chmod +x script-name.sh
```

### Run Your First Script

```bash
./script-name.sh
```

## 📝 How to Use This Repository

1. **Sequential Learning**: Follow the course index from top to bottom for a structured learning experience
2. **Topic-based Learning**: Jump to specific topics based on your needs
3. **Hands-on Practice**: Each folder contains README.md with explanations and example scripts
4. **Experiment**: Modify the scripts and observe the behavior
5. **Build Projects**: Apply concepts to create your own automation scripts
